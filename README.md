# Championz-Hackforge
ColorWise Closet is a demo AI-powered fashion assistant that recommends outfit color combinations from a user’s wardrobe using color theory. Based on skin tone and occasion, it suggests the best clothing match and explains why the combination works.
It is based on
- Age
- Gender
- Mood
- Favorite colors

Tech Stack
Frontend: Java Script , HTML ,CSS 
Backend: Python, Flask, Flash-Cors

Status
Hackathon submission – work in progress.
Frontend and backend are partially implemented.
